# MMA Manager MVP Cut (v3.1-lite)

## 0) 목적
- 4주 내에 "플레이 가능한 코어 루프"를 출시 가능한 품질로 고정한다.
- 핵심 재미를 `매니저 의사결정`으로 검증한다.
- 검증 대상:
  - 선수 1~2명 운영
  - 주간 훈련/컨디션 관리
  - 경기 시뮬레이션
  - 최소 경제 루프(흑자/적자 압박)
  - 매니저 부업 트레이드오프

---

## 1) MVP 성공 조건 (Definition of Done)
- 신규 유저가 튜토리얼 없이도 1시즌(12개월)을 진행 가능
- 파산/연패/부상 중 1개 이상 위기를 경험하고 회복 가능
- 1시즌 내 `국내 프로 랭킹 상승`을 체감 가능
- 전투 결과가 "납득 가능한 수준"으로 반복 재현 가능
- 치명 버그 없이 3시간 연속 플레이 가능

---

## 2) MVP 포함 범위 (In Scope)

### 2.1 코어 루프
- 주간 단위 진행 (월~일)
- 주간 계획:
  - 훈련 3~5일
  - 휴식 1~2일
  - 부업 0~2회
- 주간 결산:
  - 스탯 성장
  - 피로/컨디션 반영
  - 수입/지출 반영

### 2.2 선수 시스템 (축소형)
- 스탯 7종: `STR, TEC, SPD, STA, DEF, WRW, BJJ`
- 잠재력 `POT` 적용
- 성격 3종만 우선:
  - 투사형
  - 분석형
  - 스타형
- 사기(Morale) 0~100
- 나이 커브는 단순 3구간:
  - 18~26 성장 보너스
  - 27~31 보통
  - 32+ 성장 페널티

### 2.3 경기 시스템 (축소형)
- 결과 타입:
  - KO/TKO
  - Submission
  - Decision
- 3라운드 고정
- 전술 선택 3축만:
  - 타격 중심 / 밸런스 / 그래플링 중심
  - 공격적 / 밸런스 / 보수적
  - 체력 올인 / 밸런스 / 보존
- 상세 포지션(마운트/사이드 등)은 이벤트 텍스트 레벨로 축약

### 2.4 경제 시스템 (축소형)
- 수입:
  - 파이트머니
  - 스폰서(단일 등급)
  - 부업(예능/유튜브/해설 3종)
- 지출:
  - 체육관 유지비
  - 선수 월급
  - 치료비(부상 시)
- 파산 룰:
  - 잔고 0 미만 4주 지속 시 게임오버 경고 이벤트

### 2.5 부업 시스템 (MVP 핵심)
- 부업 3종만 구현:
  - 예능 출연 (2일 소모, 수익 큼)
  - 해설 게스트 (1일 소모, 수익 중간)
  - 유튜브 업로드 (1일 소모, 수익 낮음 + 인지도 성장)
- 공통 페널티:
  - 부업 수행일 훈련 효율 -15%
  - 주 3일 이상 부업 시 사기 -3

### 2.6 발굴/영입 (축소형)
- 발굴 경로 2종만:
  - 아마추어 대회
  - 지역 체육관
- 후보 3명 제시 후 1명 계약
- 계약 요소:
  - 계약금
  - 월급

### 2.7 UI
- 필수 화면:
  - 홈 대시보드
  - 선수 상세
  - 주간 계획
  - 경기 준비/결과
  - 재정 요약
- 알림:
  - 부상
  - 경기 확정
  - 계약/지출 경고

---

## 3) MVP 제외 범위 (Out of Scope)
- 멀티 선수 3명 이상 동시 관리
- 체급 변경/감량 상세 시스템
- 엘리트 은퇴 선수 컴백
- 코치/스태프 상세 트리
- 고급 스폰서 등급 체계
- 미디어/팬 세부 이벤트 체인
- 실시간 3D 액션 조작형 전투
- 고급 연출(입장, 카메라워크, 리플레이)

---

## 4) 수치 기준선 (Balance Baseline)
- 경기 길이:
  - 평균 2~3분(시뮬 연산 기준)
- 결과 분포 목표:
  - Decision 45%
  - KO/TKO 35%
  - Submission 20%
- 초반 월 손익:
  - 부업 미사용 시 소폭 적자
  - 부업 1~2회 사용 시 손익분기~소폭 흑자
- 성장 체감:
  - 6~8주 운영 시 1개 스탯에서 명확한 상승 체감

---

## 5) 4주 구현 계획

### Week 1: 코어 데이터/시간 루프
- 선수 데이터 구조 고정
- 주간 사이클 구현
- 기본 UI 골격

### Week 2: 훈련/컨디션/경제
- 훈련 실행 + 성장 반영
- 컨디션/피로/사기 반영
- 수입/지출 계산 + 재정 UI

### Week 3: 경기/랭킹/부업
- 경기 시뮬(3라운드)
- 승패 결과 + 랭킹 포인트 반영
- 부업 3종 + 트레이드오프 반영

### Week 4: 튜닝/저장/안정화
- 밸런스 1차 튜닝
- 세이브/로드(단일 슬롯 우선)
- 크래시/치명 버그 정리

---

## 6) 구현 체크리스트 (Gate)
- [ ] 1시즌 플레이 테스트 3회 완료
- [ ] 하드락/진행불가 버그 0건
- [ ] 경기 결과 로그 재현성 검증(동일 시드)
- [ ] 초반 파산 확률 20~40% 범위 유지
- [ ] 부업 선택이 승률/성장에 유의미한 영향

---

## 7) 리스크와 대응
- 리스크: 시스템 과다 연동으로 디버깅 비용 급증
  - 대응: 이벤트 버스 입력/출력 포맷 먼저 고정
- 리스크: 전투 밸런스 불안정
  - 대응: 1차는 확률표 기반, 2차에서만 상세 공식 확장
- 리스크: UI 작업량 초과
  - 대응: MVP 기간엔 텍스트 중심 UI 우선

---

## 8) 출시 후 확장 우선순위 (Post-MVP)
1. 체급/감량 고도화
2. 코치/스태프 상세 트리
3. 멀티 선수(3~6명) 확장
4. 미디어/팬/스폰서 심화
5. 3D 경기 연출 강화
